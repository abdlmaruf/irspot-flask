User Guidelines

  • Recommended browser : Google Chrome or Mozilla Firefox
  • FASTA sequence length must not be too small. (Length must be greater than 30)
  • Maximum number of characters is 50000
  • User should give a single DNA sequence in FASTA format
  • Click on the Example button for example
  • Click on the Submit button to get the prediction results
  • If your FASTA format is not valid, it will not give any prediction
  • Case-sensitive : distinguishes between lower case letter or upper case letter.
  • Visit FASTA, to know more about FASTA format.
