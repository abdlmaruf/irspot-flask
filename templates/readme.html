Readme


  • Step 1: Go to the link at http://irspotsf.pythonanywhere.com/. You will see the homepage of iRSpot-SF.

  • Step 2: Click on the Web Server tab on the navigation bar or go to this link http://irspotsf.pythonanywhere.com/server.html. You will see the server page.

  • Step 3: Once in the server page, either type or copy and paste the query DNA sequence into the input box. The input sequence should be in FASTA format. A sequence in FASTA format consists of a single initial line beginning with the symbol ">" in the first column, followed by lines of sequence data in which nucleotides or amino acids are represented using single-letter codes. Except for the mandatory symbol ">", all the other characters in the single initial line are optional and only used for the purpose of identification and description. You can input a single sequence at a time. An example sequence in FASTA format can be seen by clicking on the Example button right below the input box.

  • Step 4: Users are able to set the step size of iRSpot-SF. The predictor predicts the type (hotspot, coldspot) in each subsequence according to the step size. Example: if sequence length is 600 and step size given is 200, then the model will predict from the position 1 to 200 of the sequence whether it is hotspot or coldspot. Then it will do the same thing for the position 201 to 400 and the position 401 to 600.

  • Step 5: Click on the Submit button to see the predicted results.
